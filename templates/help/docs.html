Docs

First, create a new survey. You should see the dashboard once it has been created. You can see the code, download/view JSON, download CSV, create/rename/delete input fields, change privacy, and end the survey.

To set up your form, insert this code in your body tag:

<form id="form" action="https://brainy.ruiwenge2.repl.co/USERNAME/SURVEY/submit" method="POST">
          
  <input type="submit">

</form>

<script src="https://brainy.ruiwenge2.repl./USERNAME/SURVEY/script"></script> <!-- link the script source -->

Add input fields

You should be able to see your input fields on the top row in the table. The hashtag (#) is not an input field, it's only for the number in which a user submitted a form. For every input field, add this to the form tag:

<input name="INPUTNAME">

You can change the types of those input tags, like multiple choice or a color picker. Click here for more information.

If you don't include all the input fields you added in the dashboard, there will be an error.


Using redirecturl

The redirecturl input field allows you to redirect to a page after submitting the form, like a thank you page:

<input name="redirecturl" type="hidden" value="https://example.com/thanks"><!-- make the input hidden and redirect to https://example.com/thanks after submitting form -->

Remember, the value must be the full URL, not just /thanks.


Fetching JSON data in JavaScript

Brainy allows you to fetch JSON data, which is useful if you want to programatically find the numbers of each input field:

fetch("https://brainy.ruiwenge2.repl.co/USERNAME/{SURVEY/json").then(response => response.json()).then(response => {
  // do something with the "response" object
});
