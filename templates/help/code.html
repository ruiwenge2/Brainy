Code for the form

Hi! This page will be showing you the code for your form.

In your HTML code, use this for the form:

<form id="form" action="https://brainy.ruiwenge2.repl.co/{{username}}/{{survey}}/submit" method="POST">
  {% for i in inputfields %}
  <input name="{{i}}">
  {% endfor %}
  <input type="submit">

</form>

<script src="https://brainy.ruiwenge2.repl.co/{{username}}/{{survey}}/script"></script> <!-- link the script source -->

To use the redirecturl input field, do this:

<form id="form" action="https://brainy.ruiwenge2.repl.co/{{username}}/{{survey}}/submit" method="POST">
  {% for i in inputfields %}
  <input name="{{i}}">
  {% endfor %}
  <input name="redirecturl" type="hidden" value="https://example.com/thanks"><!-- make the input hidden and redirect to https://example.com/thanks after submitting form -->
  
  <input type="submit">

</form>

<script src="https://brainy.ruiwenge2.repl.co/{{username}}/{{survey}}/script"></script> <!-- link the script source -->

To fetch the JSON data in JavaScript, use the code below:

fetch("https://brainy.ruiwenge2.repl.co/{{username}}/{{survey}}/json").then(response => response.json()).then(response => {
  // do something with the "response" object
});
